# HOLCUS LANGUAGE CORPUS — STRUCTURED DUMPS
# Purpose: Lexical and topological language datasets for sedenion field geometry.
# Priority: EDGES and PATHWAYS over definitions.
# The 15-edge window of the 16-word sedenion increment requires
# structured relational data — hypernym, hyponym, meronym, antonym,
# cross-language concept divergence — not flat vocabulary lists.
#
# TAG SCHEMA:
# [LIVING]    — living language(s)
# [DEAD]      — extinct/classical language(s)
# [MULTI]     — multilingual / cross-language mapping
# [TOPO]      — topological/relational structure priority (edges, graphs)
# [LEXICAL]   — lexical content priority (vocabulary, definitions)
# [DUMP]      — direct downloadable bulk dump available
# [API]       — API access, may require registration
# [FREE]      — no registration required
# [REG]       — registration required, free academic use
#
# STRUCTURE: NAME | TAGS | URL | NOTES
# ============================================================

# ============================================================
# TIER 2 — CROSS-LANGUAGE CONCEPT MAPPING (TOPOLOGY PRIORITY)
# Edge divergence between languages — where concept-space folds
# ============================================================

BabelNet | [MULTI][TOPO][LEXICAL][DUMP][REG]
https://babelnet.org/download
# 500+ languages. Concept nodes with cross-language edges.
# Registration required. RDF and JSON-LD dumps available.
# The most complete cross-language topology available.
# Critical: concept divergence edges are explicit in the data structure.

ConceptNet | [MULTI][TOPO][LEXICAL][DUMP][FREE]
https://conceptnet.io/
https://github.com/commonsense/conceptnet5/wiki/Downloads
# Direct download: https://conceptnet.s3.amazonaws.com/downloads/2019/edges/conceptnet-assertions-5.7.0.csv.gz
# 35 languages. Commonsense relational edges — IsA, PartOf, UsedFor, HasA
# Edge-first structure. 34 million edges.
# Complements WordNet — different edge types, overlapping concept space

NASARI (Unified Semantic Space) | [MULTI][TOPO][DUMP][FREE]
http://lcl.uniroma1.it/nasari/
# Vector representations of BabelNet concepts — geometric embedding of edges

UWN (Universal WordNet) | [MULTI][TOPO][DUMP][FREE]
http://www.h-its.org/en/research/nlp/uwn/
# 1.5 million concepts, 200+ languages. Lightweight cross-language edge graph.
