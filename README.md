## learn()

```c
void monad_learn(Monad *m, const char *text, int verbose);
```

For each token in text:

```
idx, E  =  word_coords(token)
β[idx] +=  α × E²                             deepen at zero address
β[idx]  =  min(β[idx], β_sat)                 bound at saturation

for each co-occurring pair (i, j) in the sentence:
    A[i,j] += E_i × E_j / (|γ_i - γ_j| + GAP)    Coulomb coupling
```

`β[idx]` is monotone — it only deepens, never forgets.
`A[i,j]` is a sparse hash map keyed as `(i<<15)|j` for O(1) access (both indices < 25000 < 2^15).
GAP = 0.000707 prevents divergence when two words share a nearby zero.

The A coupling is 1/r in Riemann zero space. This is the Yang-Mills regime: turbulent, non-Abelian.

`min(w, 1/GAP)` clamps propagation. The same constant that prevents Yang-Mills collapse
in `learn()` prevents Noether cascade in `speak()`. One problem, one constant, both regulators.

## Build from Source

Dependencies: `gcc`, `make`, `libm` (standard on every *nix). Nothing else required to build.

```bash
git clone https://github.com/michaelrendier/SMMIP.git
cd SMMIP/PtolC
make
./ptolemy -h "water flows downhill"
```

The binary is statically self-contained. No install step. Run it from any directory.

**With the WordNet checkpoint** (Python 3 + NLTK, ~5 seconds):

```bash
make corpus
# downloads wordnet if needed, pipes 313K lines through ptolemy, writes monad_wordnet.bin
```

**Or point it at any corpus directly — no Python required:**

```bash
./ptolemy -l yourfile.txt
./ptolemy -l -          # stdin
./ptolemy -l /usr/share/doc/bash/README
```

Every `-l` call deepens the existing checkpoint. The checkpoint IS the model — portable, binary, self-contained.

## Checkpoint Format

Binary, little-endian. No external dependencies.

```
Header:  "PTOL"[4]  version[4]  N[4]  vocab_size[4]  A_size[4]  wc[4]  threshold[8]
Beta:    N × double       dense β field (200KB for N=25000)
Age:     N × int32        recency counters
Vocab:   vocab_size × (idx[4]  wlen[2]  E[8]  word[wlen])
A:       A_size × (i[4]  j[4]  weight[8])
```

The bundled `monad_wordnet.bin` contains 14,165 vocab entries and 766,119 A-edges from the
full WordNet 3.1 corpus. Total size: 13MB.
